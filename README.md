# .github Repository

Welcome to the `.github` repository for the [fynbosltd](https://github.com/fynbosltd) organization! This repository serves as a centralized location for GitHub-wide configurations, default community health files, and organizational standards.

## Purpose

The `.github` repository allows fynbosltd to:

- Provide organization-wide community health files such as:
  - Issue and Pull Request templates
  - Contribution guidelines
  - Code of Conduct
  - Funding information
- Ensure consistent community standards across all repositories
- Minimize duplication of common documentation in each individual repo

## What Belongs in This Repo?

This repository commonly contains:

- `.github/ISSUE_TEMPLATE/` - Issue templates
- `.github/PULL_REQUEST_TEMPLATE.md` - Pull request template
- `CODE_OF_CONDUCT.md` - Code of conduct for contributors
- `CONTRIBUTING.md` - Contribution guidelines
- `SECURITY.md` - Security policy and vulnerability disclosure guidelines
- `FUNDING.yml` - Funding links for sponsorship

**Note:** The contents of this repository are inherited by all public repositories in the fynbosltd organization unless those repositories contain their own versions of these files.

## Usage

You do not need to clone or fork this repository unless you are responsible for updating organization-wide community health files.

To contribute or suggest changes:

1. Fork this repository
2. Make your changes
3. Open a Pull Request with a clear description
